Release Notes

  • 2.0 - Implemented chat box: Choose nick when joining room. Set default nick in options. When connected to room, click speech bubble icon in upper right corner of video to toggle chat box. Switch to/from participant list at top of chat box. Click participant in list to mute/unmute them in your chat feed. Please note that this release requires a server update for the chatbox to work.
  • 1.1 - Fixed some issues with Funimation player consistently freezing up when seeking backward. It might still freeze up in some situations, but hopefully less often.
